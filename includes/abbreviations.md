*[mDNS]: Multicast DNS — local-network discovery protocol that lets devices announce a hostname like `lightnet-xxxx.local` without DNS or static IP.
*[I²C]: Inter-Integrated Circuit — two-wire serial bus (SDA + SCL) used between the controller and panels.
*[OTA]: Over-the-Air — flashing firmware wirelessly instead of plugging in a USB / programmer.
*[GPIO]: General-Purpose Input/Output — a pin on the microcontroller that can be driven high or low or read as a signal.
*[ISR]: Interrupt Service Routine — a small function the CPU jumps into when a hardware event fires.
*[PCINT]: Pin Change Interrupt — AVR feature that triggers an ISR whenever a watched pin changes state.
*[SRAM]: Static RAM — runtime memory on the microcontroller; very limited on ATmega panels.
*[LittleFS]: Little Flash File System — small filesystem the ESP uses to store scenes, palettes, and the panel firmware blob.
*[WS2812]: Single-wire addressable RGB LED used on each panel.
*[ATmega]: 8-bit AVR microcontroller family (e.g. 328P / 328PB) used in panel boards.
*[ESP8266]: Wi-Fi-enabled microcontroller — one of the two supported controller MCUs.
*[ESP32]: Wi-Fi + BLE microcontroller — the other supported controller MCU.
*[MCU]: Microcontroller — the chip running the firmware (ESP on the controller, ATmega on each panel).
*[PCB]: Printed Circuit Board — the physical board you solder components onto.
*[BOM]: Bill of Materials — the parts list for building a board.
*[WDT]: Watchdog Timer — a hardware safety timer that resets the MCU if firmware stops feeding it.
*[USBasp]: A cheap USB AVR programmer used to flash the bootloader and fuses onto a panel ATmega.
*[CRC]: Cyclic Redundancy Check — short checksum used to detect corrupted packets.
*[KMP]: Kotlin Multiplatform — Kotlin's mechanism for sharing one codebase across Android, iOS, and other targets.
*[REST]: Representational State Transfer — HTTP-style API where each URL represents a resource.
*[JSON]: JavaScript Object Notation — text-based data format used by the HTTP API.
*[NSD]: Network Service Discovery — Android's API for browsing mDNS services.
